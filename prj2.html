Stock Simulation Game

Developed alongside: Blake Montgomery & Jason Tobin

Github Link

Description

This stock simulation game was developed as part of a final group project for my CS 49J class. It build's off of a java webscrapper program developed by my classmate Jason Tobin. We used his in-progress program as the basis for this final project and expanded it into a mini simulation game.


Features

  • "Roll" for a random value to use as starting capital
  • Buy and sell (fake) stocks with in game currency (Stocks are updated in real time based on the current market)
  • Keep track of user capital and stock portfolio
